Code: How this site is built

1. Content

- All the content in Snow Before You Go is built using HTML5,
including the various pages on this site (main page, about, and this "code" page!)

2. Styling

- All the website styling is built on CSS3, fonts are pulled from Google Fonts,
dropdowns (future feature!) are from Semantic UI, and icons are from Font Awesome

3. Interaction

- All the interaction on the site (clicks, loading overlay, modal, etc) are built using JavaScript & jQuery

    jQuery plug-ins that are used (thank you!):

  • - Popup Overlay: http://dev.vast.com/jquery-popup-overlay/#basic
  • - Loading Overlay: https://gasparesganga.com/labs/jquery-loading-overlay/

4. Backend/Services

- Weather data is from the Dark Sky API, Node.js is used to bypass Dark Sky's CORS logic (sorry Dark Sky ðŸ˜‰), Heroku is used for hosting,
GitHub is used for local file hosting, Google Domains holds the snowbeforeyougo.com domain

5. Design

- Balsamiq was used for wireframing in the early days of this project,
and DesignEvo helped create the awesome "Snow Before You Go" logo!
